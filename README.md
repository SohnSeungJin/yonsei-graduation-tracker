# Yonsei Graduation Tracker Version 4.2.1

연세대학교 경영학 단일전공, 응용통계학 단일전공 또는 경영학 본전공·응용통계학 캠퍼스내 복수전공 학생이 자신의 이수 현황과 남은 졸업요건을 확인할 수 있도록 만든 비공식 웹 도구입니다.

## Version 4.2.1 추가 기능

- 계산 대상에서 `경영학 단일전공`, `응용통계학 단일전공`, `경영학 + 응용통계학 복수전공` 선택
- 응용통계학 단일전공은 전공 48학점 적용
- 응용통계학 필수학점은 2024학번 이전 18학점, 2025학번 이후 21학점 적용
- 응용통계학 본전공 기본요건에 `STA1001 통계학입문` 필수 적용
- 선택한 전공에 맞는 필수과목만 성적표에서 자동 매칭
- 성적표 검토와 `이수 과목 직접 입력`의 이수구분에 `기본 졸업요건` 추가
- 기본 졸업요건으로 분류한 과목은 교기·기초교육·대학교양·RC 세부 항목을 지정해 체크 및 학점 계산에 연결
- 요약 카드, 진행률, 남은 필수과목을 선택한 전공 유형에 맞게 표시

## 성적증명서 자동 인식 기능

- 국문 성적증명서 PDF 업로드
- 이미지형 PDF를 위한 브라우저 OCR
- 좌우 2단 성적표를 영역별로 분리해 과목명·학점·성적 인식
- 필수과목과 일부 기본 졸업요건 자동 매칭
- 자동 매칭되지 않은 과목을 특정 기본 졸업요건 항목으로 직접 분류
- 인식 결과 수정, 이수구분 선택, 3·4000단위 확인
- 확인한 과목을 기존 졸업요건 계산에 일괄 반영
- 성적표 누계학점과 인식 학점 합계를 비교해 누락 가능성 경고
- PDF와 OCR 결과를 서버에 저장하지 않는 브라우저 내 처리

## 사용 흐름

1. 계산할 전공 유형을 선택합니다.
2. 입학 학번을 선택합니다.
3. `성적증명서 자동 불러오기`에서 국문 PDF를 선택합니다.
4. OCR이 끝나면 과목명·학점·성적을 확인합니다.
5. 자동 매칭되지 않은 과목의 이수구분을 선택합니다. 기본 졸업요건은 세부 항목까지 지정합니다.
6. 전공선택·일반선택 과목은 3·4000단위 여부를 확인합니다.
7. `확인한 과목 반영`을 누르면 기존 계산 화면에 결과가 적용됩니다.
8. 누락되거나 잘못 읽힌 과목은 기존 직접입력 기능으로 보완합니다.

## 기존 졸업요건 기능

- 2018학번부터 학번별 교기·기초교육·대학교양·RC 요건 적용
- 총 126학점, 3·4000단위 45학점, 일반선택 및 전공요건 계산
- 과목별 직접 입력과 브라우저 자동 저장
- 남은 필수요건 카테고리별 표시
- 회계원리(1), 회계원리(2) 교차인정
- 졸업요건 충족 영역과 진행률 표시

## 졸업요건 기준

- [연세대학교 경영대학 졸업 요건 게시판](https://ysb.yonsei.ac.kr/board.asp?mid=m03_07)의 `경영 단일 전공`과 `캠퍼스내 복수전공` 구분을 반영했습니다.
- [연세대학교 응용통계학과 졸업이수요건 안내](https://stat.yonsei.ac.kr/stat/board/under_notice.do?article.offset=0&articleLimit=10&articleNo=220067&mode=view)의 단일전공 전공학점, 필수과목과 3·4000단위 45학점 기준을 반영했습니다.
- 응용통계학과 [수강가이드](https://stat.yonsei.ac.kr/stat/education/guide.do)의 학번별 필수과목 구성을 반영했습니다.
- 앱의 세부 학번별 기준은 현재 저장된 2018~2026학번 프로필을 사용합니다.
- 학교 규정 변경, 대체과목, 개별 승인 내역은 자동 반영되지 않으므로 연세포탈 졸업사정을 최종 기준으로 확인해야 합니다.

## 기술 구성

- React 19
- TypeScript
- Vite
- PDF.js: PDF 읽기와 페이지 렌더링
- Tesseract.js: 한국어·영어 OCR
- Vitest: OCR 텍스트 파싱과 과목 매칭 테스트

별도의 백엔드, 데이터베이스 또는 유료 OCR API를 사용하지 않습니다.

## 지원 범위와 한계

- Version 4.2.1은 연세대학교 국문 성적증명서의 현재 2단 양식을 우선 지원합니다.
- 전자 PDF와 이미지형 PDF를 처리하지만, 기울어짐·저해상도·그림자 정도에 따라 OCR 오류가 발생할 수 있습니다.
- 성적증명서에는 과목코드와 이수구분이 없으므로 자동 매칭되지 않은 과목은 사용자가 분류해야 합니다.
- 학기 구분은 졸업요건 계산에 사용하지 않습니다.
- 교환학생, 대체과목, 학과 승인 과목, 중복·재수강 등은 사용자가 확인해야 합니다.
- 현재 버전은 모바일 최적화를 핵심 범위로 삼지 않았습니다.

## 개인정보 처리

PDF는 사용자의 브라우저 안에서만 읽고 OCR합니다. 앱은 성적증명서 원본, 이름, 학번 또는 OCR 원문을 서버로 전송하거나 저장하지 않습니다. 입력한 계산 상태는 현재 브라우저의 로컬 저장소에만 저장됩니다.

## 안내

계산 결과는 참고용입니다. 최종 졸업 가능 여부는 연세포탈 졸업사정과 소속 학과 안내를 기준으로 반드시 다시 확인해야 합니다.
