# Yonsei Graduation Tracker Version 3.0

연세대학교 경영학 본전공·응용통계학 캠퍼스내 복수전공 학생을 위한 비공식 졸업요건 확인 도구입니다.

## Version 3.0 주요 기능

- 2018학번부터 학번별 교기·기초교육·대학교양·RC 요건 적용
- 총 126학점, 3·4000단위 45학점, 일반선택 및 전공요건 계산
- 과목별 직접 입력과 브라우저 자동 저장
- 남은 필수요건 카테고리별 표시
- 진행률 구간별 색상 표시
- 회계원리(1), 회계원리(2)만 교차인정
- 이용 전 유의사항과 교차인정 안내 제공

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

계산 결과는 참고용이며, 최종 졸업 가능 여부는 연세포탈 졸업사정과 소속 학과 안내를 기준으로 확인해야 합니다.
