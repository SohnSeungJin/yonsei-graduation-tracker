# Yonsei Graduation Tracker

연세대학교 경영학·응용통계학 전공 학생이 성적증명서를 바탕으로 졸업요건 충족 여부를 확인할 수 있는 비공식 웹 서비스입니다.

[서비스 바로가기](https://yonsei-biz-stat-graduation-tracker.vercel.app) · [GitHub 저장소](https://github.com/SohnSeungJin/yonsei-graduation-tracker)

> 계산 결과는 참고용입니다. 최종 졸업 가능 여부는 연세포탈 졸업사정과 소속 학과 안내를 기준으로 확인해야 합니다.

## 프로젝트 배경

졸업요건은 입학 학번과 전공 유형에 따라 달라지고, 기본요건·전공필수·전공학점·교차인정 등을 함께 확인해야 합니다. 이 프로젝트는 흩어진 기준을 하나의 계산 규칙으로 구조화하고, 학생이 성적증명서를 직접 대조하는 과정을 줄이기 위해 만들었습니다.

## 주요 기능

- 국문 성적증명서 PDF에서 과목명·학점·성적 자동 인식
- 전자 PDF와 이미지형 PDF를 위한 브라우저 OCR 지원
- 필수과목 및 일부 기본 졸업요건 자동 매칭
- 인식 결과 수정, 이수구분 재분류, 3·4000단위 확인
- 교기·기초교육·대학교양·RC 세부 요건 지정
- 경영학 단일전공, 응용통계학 단일전공, 경영학·응용통계학 복수전공 계산
- 회계원리(1)·회계원리(2) 교차인정 반영
- 입력 결과 브라우저 자동 저장 및 남은 필수요건 표시

## 지원하는 전공 유형

| 전공 유형 | 경영학 | 응용통계학 |
|---|---:|---:|
| 경영학 단일전공 | 48학점 | - |
| 응용통계학 단일전공 | - | 48학점 |
| 경영학·응용통계학 복수전공 | 48학점 | 36학점 |

공통으로 총 126학점과 3·4000단위 45학점을 확인합니다. 기본 졸업요건과 일반선택 기준은 선택한 입학 학번과 본전공에 따라 적용됩니다.

응용통계학 필수학점은 다음과 같이 구분합니다.

- 2024학번 이전: 필수 18학점
- 2025학번 이후: 필수 21학점
- 응용통계학 본전공: `STA1001 통계학입문` 기본 필수요건 적용

## 사용 방법

1. 계산할 전공 유형과 입학 학번을 선택합니다.
2. 성적증명서 PDF를 첨부합니다.
3. 자동 인식된 과목명·학점·성적을 검토합니다.
4. 미분류 과목의 이수구분과 기본 졸업요건 세부 항목을 지정합니다.
5. 결과를 반영한 뒤 남은 졸업요건과 학점 진행률을 확인합니다.
6. 누락되거나 잘못 인식된 과목은 직접입력으로 보완합니다.

## 구현 포인트

- 입학 학번과 전공 유형별 졸업요건을 데이터 프로필로 분리했습니다.
- OCR 결과를 바로 계산하지 않고 사용자가 검토·수정하는 단계를 두었습니다.
- 필수과목, 기본요건, 전공선택, 일반선택을 구분해 중복 학점 반영을 방지했습니다.
- PDF 원본과 OCR 결과는 외부 서버로 전송하지 않고 사용자의 브라우저에서 처리합니다.

## 기술 구성

- React 19, TypeScript, Vite
- PDF.js, Tesseract.js
- Vitest, Oxlint
- Vercel

별도의 백엔드, 데이터베이스 또는 유료 OCR API를 사용하지 않습니다.

## 지원 범위와 한계

- 현재 2018~2026학번과 연세대학교 국문 성적증명서의 2단 양식을 우선 지원합니다.
- 스캔 품질, 기울어짐, 그림자에 따라 OCR 결과가 달라질 수 있습니다.
- 성적증명서에 이수구분이 없는 과목은 사용자가 직접 분류해야 합니다.
- 대체과목, 교환학생 학점, 개별 승인 과목, 재수강 및 중복 이수는 사용자가 확인해야 합니다.
- 학교 규정 변경은 자동 반영되지 않습니다.

## 개인정보 처리

성적증명서 원본, 이름, 학번, OCR 원문은 서버에 업로드하거나 저장하지 않습니다. 계산에 반영한 입력값만 현재 브라우저의 로컬 저장소에 저장됩니다.
